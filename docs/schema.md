# DB Schema

| Table           | Key columns                                                                                                                                                                                                                               |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `users`         | id, email, passwordHash                                                                                                                                                                                                                   |
| `userSettings`  | userId (PK, FK) ŌĆö cuisinePrefs[]/dietaryRestrictions[] (defaults new plans inherit), calorieTarget/proteinTarget/carbsTarget/fatTarget (nullable; NULL ŌåÆ global default). 1:1 with users, created on first save                           |
| `sessions`      | id (PK), userId FK, expiresAt                                                                                                                                                                                                             |
| `meals`         | id, userId FK (NULL = global/shared, else personal), name, calories, proteinG, carbsG, fatG, tags[], allowedSlots[] (empty = any slot), imageUrl, description, ingredients[], instructions, timeMinutes, difficulty, servings (default 1) |
| `plans`         | id, userId FK, name, weekStart, cuisinePrefs[], dietaryRestrictions[]                                                                                                                                                                     |
| `weekSlots`     | (planId, date, mealType) composite PK, mealId FK                                                                                                                                                                                          |
| `slotRepeats`   | (planId, mealType) composite PK, groupBreaks[6] booleans (gaps Mon\|Tue..Sat\|Sun; true = split, false = joined) ŌĆö weekly repeat pattern per meal type; no row = every day independent                                                    |
| `mealFavorites` | (userId, mealId) composite PK, both FK cascade ŌĆö per-user bookmark on a meal                                                                                                                                                              |
